Implement the output module that writes scrape results to timestamped JSON files with incremental saving support.

Purpose: Users need their scraped profile URLs persisted to disk in a predictable, timestamped format. The output module is the first half of the delivery pipeline — without it, scraped data is lost when the process exits.

Output: src/output.ts with generateOutputPath, ensureOutputDir, saveUrlsToFile, createIncrementalSaver functions. Updated src/types.ts with OutputOptions and IncrementalSaverOptions interfaces. Test file src/output.test.ts.
</objective>

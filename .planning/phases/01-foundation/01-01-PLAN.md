## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Config file → Application | External JSON config loaded and parsed; malicious config could inject unexpected values |
| CLI args → Application | User-provided arguments parsed and validated; injection possible via malformed input |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-01-01 | Tampering | config loading | medium | mitigate | Zod schema validation rejects invalid config structures; cosmiconfig only loads known file formats |
| T-01-02 | Tampering | CLI arguments | low | mitigate | yargs type validation ensures arguments match expected types |
| T-01-03 | Spoofing | npm packages | high | mitigate | Package legitimacy checkpoint (blocking human verify) for cosmiconfig and @biomejs/biome before installation |
| T-01-04 | Info Disclosure | config files | low | accept | Config files may contain webhook URLs; accept risk as project is internal tool |
</threat_model>

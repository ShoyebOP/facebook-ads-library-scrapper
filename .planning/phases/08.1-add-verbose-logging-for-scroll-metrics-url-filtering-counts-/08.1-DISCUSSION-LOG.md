## Log level for metrics

| Option | Description | Selected |
|--------|-------------|----------|
| INFO level (Recommended) | Visible by default. Metrics show up in normal output — good for monitoring production runs. | ✓ |
| DEBUG level | Hidden unless LOG_LEVEL=debug. Keeps normal output clean, metrics available when debugging. | |
| Both levels | Summary at INFO, detailed breakdown at DEBUG. Gives visibility without noise. | |

**User's choice:** INFO level (Recommended)
**Notes:** User wants metrics visible by default without enabling debug mode.

## Metrics detail level

| Option | Description | Selected |
